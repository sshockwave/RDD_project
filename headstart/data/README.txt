README FILE -- Replication Materials

Paper:  Regression Discontinuity Designs Using Covariates, Review of Economics and Statistics, forthcoming.
Authors: Calonico, Cattaneo, Farrell and Titiunik
Date: 07-MAY-2018

Software needed for replication can be found here: https://sites.google.com/site/rdpackages/

Latest version of files can be found here: https://sites.google.com/site/rdpackages/replication/

Files included:

(1) headstart.csv: CSV (Comma Separated Value format) dataset. 
(2) headstart.dta: DTA (Stata format) dataset.
(3) Calonico-Cattaneo-Farrell-Titiunik_2018_RESTAT.do: Stata replication do-file.
(4) Calonico-Cattaneo-Farrell-Titiunik_2018_RESTAT.R: R replication script.
